## 1. Smart Purchase Reorder System (Full Stack Web Application)

### 🧠 Idea & Planning

The goal of this project was to automate inventory management and ensure timely reordering of products. Manual inventory tracking can lead to stock shortages or overstocking, so I designed a system that monitors stock levels and triggers reorder actions automatically.

---

### ⚙️ System Design

The system consists of:

* Database to store product details, stock levels, and purchase records
* Backend logic implemented using Django
* Frontend interface built using HTML and CSS
* Automated logic for detecting low stock and generating reorder alerts

---

### 🔄 Development Process

1. Designed database models for products, stock levels, and purchase records
2. Implemented backend logic using Django views
3. Created user interfaces for managing inventory
4. Developed logic to monitor stock levels and trigger reorder alerts
5. Integrated backend with frontend using context variables
6. Tested system functionality and fixed bugs

---

### 🚧 Toughest Problems Solved

#### Problem 1: Low-stock detection logic

Ensuring accurate detection of low inventory levels in real time was challenging.

✔ Solution:
Implemented conditional logic to continuously monitor stock levels and trigger alerts when thresholds are reached.

---

#### Problem 2: Data consistency during updates

Maintaining correct stock values during multiple transactions was difficult.

✔ Solution:
Handled database updates carefully and validated data at each step.

---

### 🐞 Debugging & Learning

During development, I faced issues related to incorrect stock updates and UI data display. I resolved them by testing different scenarios and debugging step-by-step.

---

### 📌 Key Learnings

* Gained experience in building inventory management systems
* Improved backend logic and database handling
* Learned how to implement automated alert systems

---

### 📌 Outcome

A functional system that efficiently manages inventory, detects low stock, and supports automated reorder decisions.
